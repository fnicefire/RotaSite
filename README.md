## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/fnicefire/RotaSite
cd RotaSite
```

Install dependencies:

```bash
sudo apt install nginx
```

## ▶️ Usage

Use NGINX to host it on `/var/www/rotasite`
